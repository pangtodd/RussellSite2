#### By Todd Pangilinan

updated website for author Mark Russell

## Technologies used:

* 11ty
* Decap (CMS)
* Cloudinary (image hosting)
* HTML
* CSS
* Bootstrap
* markdown

## Description: updated website (CMS capabilities) for author client

building upon the static site I previously built, I wanted to update the site to provide a CMS/interface for the user to updaate content. Prioritizing simpilicity and cost (free), I decdided to use 11ty for content and Decap for a CMS. I added Cloudinary for the image server. I set up a blog portion, but the author isn't ready for that- still useful to update other content, like the upcoming events/annoucements and book content.
